We believe development should be **instant, collaborative, and intelligent**. CodeVerse achieves this by treats the codebase not as a set of files, but as a living "Neural Workspace" where AI and humans move in synchronized harmony.

## 🚀 Key Features (God-Level Tier)

### 🧠 Neural Architect (AI Core)
The CodeVerse AI doesn't just suggest lines; it understands your **entire project DNA**.
- **Global Context Ingestion**: Uses a recursive-context scanner to feed your whole project into the LLM.
- **Architectural Reasoning**: Performs multi-file logic checking and visual CSS consistency audits.
- **Local Sovereignty**: Powered by **Ollama (CodeLlama)** for zero-latency, private-first intelligence.

### ⏳ Chronos History (Time Mastery)
Time is fluid in CodeVerse. Never fear a breaking change.
- **Side-by-Side Diff Engine**: High-fidelity Monaco comparison view for every snapshot.
- **Atomic Reverts**: Restore previous project states instantly across the entire distributed workspace.
- **Visual Timeline**: An intuitive UI to scrub through your project's evolution.

### 📡 Aegis Deployment Bridge
CodeVerse bridges the "Local-to-Live" gap with one click.
- **Instant Propagation**: Propagate your project to the public internet via the **Aegis Secure Tunnel**.
- **Edge Static Serving**: Dedicated secondary server on port 5001 handles global asset delivery.
- **Real-World URLs**: Get a live `https://codeverse-*.loca.lt/` URL for stakeholder reviews instantly.

### 🧑‍🤝‍🧑 Neural Presence
Collaboration so smooth it feels like the person next to you.
- **Ghost Cursors**: Fluid peer movement with sub-millisecond sync.
- **Living Status**: Dynamic activity tags (e.g., *"Optimizing main.js"* or *"Reviewing style.css"*).
- **Socket Fusion**: Built on a high-throughput Socket.io backbone for zero-lag interaction.

## 🛠️ Installation & Neural Sync

### 1. Initialize the Core
```bash
git clone https://github.com/your-username/codeverse.git
cd codeverse
```

### 2. Configure the Neural Grid
Create a `/server/.env` with these critical tokens:
```bash
PORT=5000
DEPLOY_PORT=5001 # Dedicated port for Aegis Bridge
MONGO_URI=your_mongo_path
GITHUB_CLIENT_ID=your_id
GITHUB_CLIENT_SECRET=your_secret
```

### 3. Ignite the Engine
```bash
# Install Aegis Requirements
npm install -g localtunnel

# Launch Services
npm run dev # Run from root if workspace-enabled, otherwise in /client and /server
```
