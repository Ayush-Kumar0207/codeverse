# Code of Conduct

CodeVerse contributors and community members must provide a welcoming, professional, and harassment-free environment.

## Expected behavior

- Be respectful, constructive, and specific.
- Welcome different experience levels and perspectives.
- Critique ideas and code without attacking people.
- Protect private information and disclose security concerns responsibly.
- Accept maintainer decisions made to protect users and the project.

Harassment, discrimination, threats, sexualized conduct, doxxing, deliberate disruption, or publishing another person's private information are not tolerated.

Report conduct concerns privately through the maintainer's GitHub profile or repository security-advisory channel. Maintainers may edit or remove content and temporarily or permanently restrict participation when necessary.
